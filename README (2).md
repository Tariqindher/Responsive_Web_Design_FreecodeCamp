# 📅 Day 2 – 22-06-2026

## 📚 Topics Learned

### 🔹 Div Element

A `<div>` is a container used to group HTML elements together and organize webpage sections.

### 🔹 ID Attribute

An `id` uniquely identifies a single HTML element on a webpage. Each id should be used only once.

### 🔹 Class Attribute

A `class` is used to group multiple HTML elements together. Multiple elements can share the same class.

### 🔹 HTML Entities

HTML entities are special codes used to display reserved characters in HTML.

**Examples:**

* `&lt;` = <
* `&gt;` = >
* `&copy;` = ©

### 🔹 Script Element

The `<script>` tag is used to add JavaScript to a webpage or link an external JavaScript file.

**Example:**
`<script src="script.js"></script>`

🖼️ Image Formats
PNG → lossless (high quality)
JPG → lossy (compressed)
WEBP / AVIF → modern, optimized formats
🗜️ Compression

Reduces file size to improve website speed.

⚖️ Image Licenses
©️ Copyright → protected content
🚫 All Rights Reserved → permission needed
🧠 Fair Use → limited use (review/parody)
📜 Creative Commons → free with conditions
🌍 CC0 / Public Domain → free use
🖼️ Unsplash / Pixabay → free image sources
🎨 SVG (Scalable Vector Graphics)
📐 SVG → vector-based image
🟦 Raster → PNG/JPG (pixel-based)
🚫 Raster zoom → blurry

fill → color
stroke → border
